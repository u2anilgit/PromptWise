rewrite_prompt
Strip filler, apply role framing
Verbose prompts with pleasantries cost real money on every single call.
Before (89 tokens): "I was wondering if you could perhaps help me rewrite this code to be more Pythonic. I'd really appreciate it if you could focus on readability." After (42 tokens): "From a software engineering perspective, rewrite for Pythonic style and readability." Saves: 53%
optimize_context
Compress document to token budget
Large docs break context limits. Score, rank, and drop lowest-value chunks.
Input: 8,000-token legal agreement Budget: 2,000 tokens → Drops: appendices, boilerplate, examples → Keeps: terms, liability clauses, payment Saves: 75% · chunks_dropped: 6
plan_cache
Design cache breakpoints
Same system prompt sent 10× per day? Pay once, reuse many times.
Messages: system(2K) + doc(8K) + user(500) Reuse: 10× → Cache system at index 0 (TTL: 1h) → Cache doc at index 1 (TTL: 1h) Saves: 77% on subsequent turns
batch_prompts
Merge 2–5 tasks into one prompt
Separate calls share no context. Batch related tasks to eliminate setup tokens.
Task 1: "Summarize this email" Task 2: "Extract sender and date" Task 3: "Flag urgent items" → Merged with semantic connectives Saves: ~45% vs. 3 separate calls
summarize_thread
Compress conversation for handoff
Context window filling up? Distill the session into a compact reset_prompt.
Input: 40-turn conversation (18,000 tokens) Output (800 tokens): "Decided: PostgreSQL. Rejected Redis (no relations). Implemented: user model, migrations. Auth pending. Next: JWT middleware + refresh token rotation." Saves: 95% · ready for new session
